# Vividamente

O **Vividamente** e uma plataforma de jogos cognitivos com foco na estimulacao mental de idosos, usando jogos interativos e um sistema de avaliacao em escala Likert.

## Objetivo

O projeto busca auxiliar no estimulo de:
- memoria
- atencao
- raciocinio logico
- concentracao
- interacao cognitiva

Tambem coleta avaliacoes para analisar acessibilidade, usabilidade e experiencia geral durante o uso.

## Jogos Disponiveis

- Jogo da Memoria
- Genius
- Damas
- Caca-palavras
- Palavras Cruzadas
- Jogo da Velha

## Sistema de Avaliacao

A plataforma possui uma area de avaliacao baseada na **Escala Likert**, com 21 perguntas.

| Valor | Resposta |
|------|------|
| 1 | Discordo totalmente |
| 2 | Discordo |
| 3 | Neutro |
| 4 | Concordo |
| 5 | Concordo totalmente |

As respostas sao armazenadas em banco MySQL de forma anonima.
